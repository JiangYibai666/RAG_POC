## 1. 在 CLI 里输入调查语句

启动后你会看到 `> ` 提示符，直接输入自然语言问题，例如：

```text
User U002 sold a Tesla Model X for $1,000,000
```

或者中文也可以（建议先用英文模板，命中 mock 数据更稳定）：

```text
请调查用户 U002 以 1000000 美元出售 Tesla Model X 是否异常
```

## 2. 观察多 Agent 协作过程

你会看到类似这样的流式过程：

- HostAgent 解析意图
- 调用 MarketAgent 做价格异常判断
- 调用 TransactionAgent 做历史与关联方分析
- 最后输出结构化风险结论（风险等级、证据链、建议动作）

## 3. 继续追问 / 结束会话

- 想继续：再输入下一条 query（可以换用户、换资产、换金额）
- 退出：输入 `exit`

---

你可以先跑这 3 条经典测试语句（逐条输入）：

```text
User U002 sold a Tesla Model X for $1,000,000
User U001 sold an iPhone 15 Pro for $900
Check whether user U003 may be doing structuring transactions
```

一般预期：

- 第一条会偏高风险（价格极端偏离 + 可疑交易模式）
- 第二条通常低风险
- 第三条会触发交易模式分析
